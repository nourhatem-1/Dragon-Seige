# Dragon Siege

Dragon Siege is a kids' castle-defense game where an automated castle fights a dragon using Fire Balls and Freeze Balls.

## Features
- Ball lifting mechanism
- Color-based sorting system
- Solenoid shooter
- Servo aiming system
- ESP32 dual-controller architecture
- Bluetooth communication

## Hardware
- 2 × ESP32
- TMC2208 Stepper Driver
- NEMA17 Stepper Motor
- SG90 Servos
- TCS34725 Color Sensor
- Solenoid Launcher
- IR Sensors
- LCD Display
